# CodeSyntaxInstitute

Eine moderne Webseite für das CodeSyntaxInstitute - Ihre Programmierschule für die Zukunft.

## Übersicht

Diese Webseite präsentiert das CodeSyntaxInstitute, eine Programmierschule, die praxisnahe Ausbildung in verschiedenen Programmiersprachen und Technologien anbietet.

## Funktionen

- **Responsive Design**: Optimiert für Desktop, Tablet und Mobile
- **Übersichtliche Navigation**: Schneller Zugriff auf alle Bereiche
- **Kursübersicht**: Präsentation der verfügbaren Programmierkurse
- **Kontaktformular**: Direkte Kontaktmöglichkeit für Interessenten

## Struktur

```
CodeSyntaxInstitute/
├── index.html    # Hauptseite
├── styles.css    # Stylesheet
└── README.md     # Dokumentation
```

## Verwendung

Öffnen Sie `index.html` in einem Webbrowser, um die Webseite anzuzeigen.

## Technologien

- HTML5
- CSS3 (mit CSS Custom Properties)
- Responsive Design mit CSS Grid und Flexbox

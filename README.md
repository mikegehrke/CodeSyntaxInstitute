# Code Syntax Institute

A modern, professional, responsive website for the Code Syntax Institute featuring animated sections, 3D backgrounds, and AI-based interactive effects.

## Features

- **3D Animated Background**: Interactive particle system with mouse-responsive effects
- **AI Bubbles Animation**: Floating bubble animations for an interactive feel
- **Responsive Design**: Optimized for all screen sizes
- **Modern UI**: Clean design with gradient colors and smooth animations

## Project Structure

```
root/
├── index.html              # Homepage with animated sections
├── about.html              # About page with institute information
├── styles/
│   ├── global.css          # Base styling common to all pages
│   ├── home.css            # Homepage specific styling
│   └── animations.css      # Keyframe animations
├── scripts/
│   ├── main.js             # General site functionality
│   ├── ai-bubbles.js       # AI bubble animations
│   └── three-background.js # 3D particle background
├── assets/
│   ├── logo.png            # Site logo
│   ├── bg.jpg              # Background image
│   └── icons/
│       ├── icon-ai.svg     # AI icon
│       ├── icon-code.svg   # Code icon
│       └── icon-learn.svg  # Learn icon
└── README.md
```

## Deployment

This site is ready for GitHub Pages deployment. The `index.html` serves as the entry point.

## Technologies

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
- Canvas API for 3D effects
